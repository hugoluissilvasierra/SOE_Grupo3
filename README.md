### Clonar el repositorio
`$ git clone https://github.com/hugoluissilvasierra/SOE_Grupo3.git`

`$ git checkout -b feature/NombreApellido`

`$ touch nombreApellido.html`

`$ git add .`

`$ git commit -m 'Inicial commit Nombre Apellido'`
